# 子供向け掛け算アプリ

react-hooks の勉強を兼ねて、子供向け計算アプリを作成します。
![](/README-images/完成イメージ.png)

## 機能要件

- 問題がランダムに抽出され、終了するまで繰り返し練習が可能。
- 回答ボタンを押下すると正誤判定される。
- 解いた履歴が閲覧できる。
- 解いた履歴から、正答率が確認できる。

## ルール

- React Hooks を使ってモダンなコードで開発する。
- マジックナンバーは使用しない。
- action は定数を利用する。
- 可読性を重視しながら開発する。
- 今後の追加機能開発のため、しっかりとリファクタリングもしていく。
- UI はこだわりすぎない。
